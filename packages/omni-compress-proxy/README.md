# @dharanish/omni-compress (deprecated)

This package has been renamed to [`omni-compress`](https://www.npmjs.com/package/omni-compress).

Please update your dependency:

```bash
npm install omni-compress
```

```bash
# bun
bun add omni-compress

# pnpm
pnpm add omni-compress

# yarn
yarn add omni-compress
```

And update your imports:

```diff
- import { compressImage } from '@dharanish/omni-compress';
+ import { compressImage } from 'omni-compress';
```

This package re-exports everything from `omni-compress` verbatim and will continue to work, but it will not receive documentation or changelog updates. All future development happens in [`omni-compress`](https://www.npmjs.com/package/omni-compress).
